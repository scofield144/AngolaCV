# **App Name**: AngolaCV Pro

## Core Features:

- High-Pattern CV Creation & Download: Create CVs using localized Angolan templates, mirroring Europass structure but with Angolan tweaks.
- Secure Storage on Platform: Securely store CVs on the platform with opt-in privacy controls.
- Search by Categories/Region/Fields: Enable users to make their profiles visible for search by categories, region (Angolan provinces), and fields, with user-controlled visibility.
- Shareable Links with Tracking: Generate shareable links for CVs with customizable URLs and view tracking.
- Social/Linktree Integration: Integrate social and personal links (LinkedIn, YouTube, etc.) into profiles as a visual hub.
- AI Assistance & Scoring: AI-powered tool that suggests content based on Angolan job trends and scores CVs for ATS compatibility.
- Cover Letter & Portfolio Builder: Create cover letters and upload portfolios to complement CVs.

## Style Guidelines:

- Primary color: Saturated orange (#E67E22) to evoke a sense of energy, creativity, and warmth, referencing the vibrant Angolan landscape. 
- Background color: Desaturated light orange (#F2D7B7), for a clean, bright, and professional feel.
- Accent color: Analogous yellow (#F1C40F) for calls to action and highlighting important elements, bringing brightness.
- Body and headline font: 'PT Sans', a humanist sans-serif that blends a modern look with warmth, appropriate for both headlines and body.
- Professional, clear icons to represent skills, industries, and social media links.
- Clean and structured layout that adapts to different CV templates and sections.
- Subtle transitions and loading animations to provide a smooth user experience.
